Installation
============

Install PyKinGenie using uv:

.. code-block:: bash

   uv pip install pykingenie

Or for development:

.. code-block:: bash

   git clone https://github.com/osvalB/pykingenie.git
   cd pykingenie
   uv pip install -e ".[dev]"

Installation
============

Install PyKinGenie using pip:

.. code-block:: bash

   pip install pykingenie

Or for development:

.. code-block:: bash

   git clone https://github.com/osvalB/pykingenie
   cd pykingenie
   pip install -e ".[dev]"
